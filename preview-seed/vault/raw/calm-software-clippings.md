# Calm Software Clippings

Reference: [Craig Mod - Calm Technology](https://craigmod.com/essays/calm_technology/).

Calm software avoids false urgency, makes state legible, and gives users a sense that the tool will still be useful tomorrow. The article framed restraint as a product feature, not an aesthetic flourish.

Useful product translation:

- Reduce performative loading states.
- Make empty screens feel intentional instead of abandoned.
- Prefer quiet confidence over busy reassurance.

This is a helpful companion to [[Warm Product Language]] and [[Trust Signals]].
