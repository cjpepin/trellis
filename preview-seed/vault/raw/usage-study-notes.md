# Usage Study Notes

Reference: [Nielsen Norman Group - First-Time User Experience](https://www.nngroup.com/articles/first-time-user-experience/).

Repeat usage rose when the second visit started from something the first visit left behind. Users were much more likely to return when they could open the app and immediately see a durable artifact: a note, a linked cluster, or a short summary that felt worth continuing.

Observed behaviors:

- Empty states created hesitation unless the app suggested one clear next action.
- The graph became reassuring only after notes were dense enough to feel lived in.
- People described the product as smart only after they could revisit the residue of prior chats.

This file is a good counterweight to feature-sprawl conversations because it keeps attention on repeatable value, not novelty.
